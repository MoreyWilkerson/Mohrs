# Mohrs

Mohrs Cirle GUI through TKINTER and matplotlib. Plots the values calculated dynamically for Mohrs circle.

If you are new to coding and would like to use this -
Download Python -- https://www.python.org/downloads/
Download VSCODE -- https://code.visualstudio.com/Download

You will need to download a library for python. 
to do this, press the windows key and type in terminal. in the terminal type
pip install Matplotlib
and press enter. 

open the program file on VSCODE and run. it should run fine.
